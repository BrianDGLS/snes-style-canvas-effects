Snes Style Canvas Effects

Here are three atmospheric Super Nintendo, SNES, style effects.

All three are written in Typescript and compile to around ~100 lines of js code. There were no external libraries used. I wanted to give an example of how to use the canvas api without relying on a third party library.

Feel free to view the source code on github. I'd also really appreciated it if you could star the repo.
